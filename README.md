# Arquitetura Rest 🔥🏆

Code 👨‍💻

Iniciando o Projeto

    yarn init 

Instalando Typescript

    yarn add typescript
    |
    tsc --init
    |
    yarn add typescript -D
    |
    yarn add @types/node -D

Para criar o servidor

    yarn add express
    |
    yarn add @types/express -D

Para ficar atualizando o servidor

    yarn add ts-node-dev -D
